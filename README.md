## GERS Frontend

## Description :scroll:

    Government Enterprise Repository System

## Dependency

-   Tools and technologies

```
    React js
    Redux
    Material ui
    Bootstrap
```

## Getting started

-   Clone the project

```
    > git clone <url>
```

-   Navigate to the project repository

```
    > cd GERS-frontend
```

-   Generate .env

```
    > copy .env.example .env
```

-   Install

```
    > yarn install or npm install
```

-   Run

```
    > yarn start or npm start
```
